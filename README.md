# 不同語言表達對生成式AI撰寫程式之可讀性探究調查

大家好，我是台南一中高三的陳秉宏，這份調查是關於我的個別研究，針對AI生成程式碼的可讀性進行探討。

## 使用說明

### 受試者條件

只要各位熟悉c++語法以及競技程式相關的基礎演算法，都歡迎填寫。但是請先確定自己熟悉關於此份調查的7個題目，以及相關的語法和演算法。在data資料夾中可看到七個小資料夾，題目和該題的4個程式碼都在各自的資料夾裡面。可以先確認自己會不會相關的語法和演算法並查資料，如果認為自己學習後仍不夠熟悉或是不會的話請不要填寫這份問卷(務必7題都要完全填答或不要填寫這份問卷，不要只填某幾題)。

以下是本人認為每一題所需的相關知識(僅供參考，仍需以個人狀況為主)

* Problem1 : for+while迴圈
* Problem2 : getline+string基本操作
* Problem3 : stack
* Problem4 : 一點點數學+char基本操作
* Problem5 : 對答案二分搜
* Problem6 : LCS/LIS
* Problem7 : 時光倒流+floyd warshall (可以參考[TOJ1009](https://toj.tfcis.org/oj/pro/1009/))

本調查共有7題以及28個程式碼，程式碼皆已確認可正確答題，請受試者務必耐心填答，感謝配合。

### 填寫教學

表單的第一頁為基本資料填寫，僅為確認填寫者是否惡意或重複填寫，並無其他個資上的使用(若為社會人士請忽略學校與年級的部分)。

第二頁開始為題目調查，每一頁一題。請先點入github上相對應的資料夾並閱讀題目敘述和4個程式碼，並針對每一個程式碼進行可讀性的評分，可讀性越佳分數越高(包括但不限於程式碼的撰寫架構、註解使用、變數名稱)，最低0分，最高10分，以0.5分為單位(如:8.5、7、6.5等等)，如有針對程式碼的其他發現或加以說明，請在每頁最下面的「其餘問題與感想」留下想法，並明確標記針對哪一份程式碼有什麼樣的問題或感想。
